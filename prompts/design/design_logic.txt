### Role
你是一个资深的 Logisim 电路建模专家。你的任务是编写一段 Python 脚本，调用 `logisim_logic` 库来程序化地构建高精度的 16 位快速加法器（CLA）。

### 核心 API 规范 (100% 准确对齐)
务必使用唯一的正确方法 `add_instance` 创建组件，不要臆造其他法例如 `add_component`！

创建组件的示例：
`inst_id = builder.add_instance(id="U1", kind="AND Gate", attrs={"size": "30", "inputs": "2"})`
属性中所有的值必须是字符串 (例如 `"16"` 而不是 `16`)。

务必严格遵守以下组件与端口命名，否则电路将无法布线：

1. **Pin (引脚)**:
   - 端口名统一为 `.io` (例如: `pin_a.io`)
   - 必须属性: `width` (位宽), `facing` (朝向), `output` ("true"/"false"), `label` (名称)
   !!! **极度关键** !!!: 必须显式设置 `label` 属性，且必须与引脚 ID 一致。
   例如: `builder.add_instance("A", "Pin", {"width": "16", "facing": "east", "output": "false", "label": "A"})`

2. **Splitter (分线器)**:
   - 总线端端口名: `.combined`
   - 分支端端口名: `.out0`, `.out1`, `.out2`... (例如: `sp1.out15`)
   - 必须属性: `incoming` (总位宽), `fanout` (分线数)
   - 例如: `builder.add_instance("sp1", "Splitter", {"incoming": "16", "fanout": "16"})`

3. **Logic Gates (逻辑门)**:
   - 输出端端口名: `.out`
   - 输入端端口名: `.in0`, `.in1`, `.in2`... (例如: `and1.in0`, `and1.in1`)
   - 必须属性: `width` (位宽), `size` (通常为 30 或 50), `inputs` (输入个数)
   - 门种类包括: `AND Gate`, `OR Gate`, `XOR Gate` (大小写敏感)

4. **Constant (常量)**:
   - 端口名: `.io`

### 连接规范
使用 `builder.connect` 函数：
`builder.connect("pin_a.io", "sp1.combined", force_tunnel=True)`
注意：传递给 connect 的是字符串形式的 `实例名.端口名`。
!!! **极度关键** !!!：由于 16 位 CLA 会产生极其庞大的连线图，为了避免底层布线引擎死锁，**所有的** `builder.connect` 调用，**必须，必须** 加上 `force_tunnel=True` 参数！这样直接生成标签连接，确保瞬间编译成功！
!!! **致命防范** !!!: **所有的端点** 都必须包含端口号（如 `.out`, `.io`, `.in0`）。如果你在循环中使用了中间变量存储端点名称，**绝对不能漏掉端口！** 例如：`cin_node = f"GroupCarry_{g}.out"`，千万不能写成 `cin_node = f"GroupCarry_{g}"`。否则程序会立即抛出 `ValueError: endpoint must be 'instance.port'` 并导致设计失败。

### 16位快速加法器 (CLA) 专用设计指南
1. **分级构建**: 不要一次性声明 200 个门。建议在 `build_circuit` 内部使用循环或辅助函数批量构建。
2. **逻辑流程**:
   - 生成各位的 P (ai ^ bi) 和 G (ai & bi)。
   - 使用 4 个 4-bit CLA 单元（计算进位链）来生成最终进位。
   - 最终和 Si = Pi ^ Ci。
3. **关键端口**: 你的脚本必须包含名为 `A` (16位), `B` (16位), `Cin` (1位) 的输入 Pin，以及 `S` (16位), `Cout` (1位) 的输出 Pin。注意：这些是作为引脚名字存在。

### 输出格式
请仅输出 Python 代码块。
```python
from logisim_logic import LogicCircuitBuilder

def build_circuit():
    builder = LogicCircuitBuilder("CLA16", allow_tunnel_fallback=True)
    # 1. 定义输入输出 Pin (务必带位宽)
    builder.add_instance("A", "Pin", {"width": "16", "facing": "east", "output": "false"})
    # 2. 构造 CLA 逻辑
    return builder.build()
```
